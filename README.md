# House Price Predictor

A Machine Learning project that predicts house prices using historical housing data.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Features

- Data preprocessing
- Data visualization
- Linear Regression model
- House price prediction
- Model saving using Joblib
